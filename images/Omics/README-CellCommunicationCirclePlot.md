## Download Instructions

The example image for the Cell-Cell Communication Circle Plot tutorial needs to be downloaded manually and placed in this directory as `CellCommunicationCirclePlot_demo.png`.

**Image Source:** https://github.com/user-attachments/assets/43e63ab6-0ba8-4547-a84c-fa116f314f5a

## Manual Download Steps

1. Visit the URL above in a web browser
2. Right-click on the image and select "Save Image As..."
3. Save the file as `CellCommunicationCirclePlot_demo.png`
4. Place the file in the `images/Omics/` directory

The image shows an example of a cell-cell communication network circle plot with different cell types (B cells, CD8 T, CD14+ Mono, NK, DC, etc.) connected by colored chords representing communication pathways.
